# 图片预览器 (Image Preview Extension)

一个轻量级、高性能的 Chrome 网页图片预览插件。支持在任何网页上快速全屏预览图片，并提供原生级丝滑的缩放、拖拽和旋转体验。

## ✨ 核心特性

- **🔍 丝滑缩放**：支持以鼠标为中心的滚轮缩放，带有物理惯性缓动，彻底告别卡顿。
- **🖱️ 自由拖拽**：全屏模式下可任意拖拽移动图片位置，零延迟跟手。
- **🔄 图片旋转**：支持 90 度步进的左转和右转，自动适配视口尺寸。
- **⚡ 极致性能**：
  - 采用 `transform` 硬件加速，彻底避免重排 (Layout Thrashing)。
  - 引入 `requestAnimationFrame` 进行高频事件（如滚轮缩放）的渲染节流。
  - GPU 友好的阴影渲染 (`box-shadow` 替代了高消耗的 `filter`)。
- **🛡️ 无痕注入**：使用 Shadow DOM 隔离 UI 组件，绝对不污染宿主网页的任何 CSS 样式。
- **⌨️ 快捷键支持**：支持丰富的键盘快捷键操作，体验更高效。

## 🚀 安装指南

由于本插件目前可通过本地开发模式进行加载：

1. 下载或克隆本仓库到本地。
2. 打开 Chrome 浏览器，访问扩展程序管理页面：`chrome://extensions/`
3. 在页面右上角开启 **“开发者模式”**。
4. 点击左上角的 **“加载已解压的扩展程序”** 按钮。
5. 选择本插件所在的文件夹（即包含 `manifest.json` 的目录）。
6. 安装完成！刷新任意网页即可体验。

## 📖 使用说明

### 🖱️ 鼠标操作
- **悬停预览**：将鼠标悬停在网页上的任意图片，图片右上角会自动出现“预览”按钮，点击即可进入全屏预览模式。
- **滚轮缩放**：在预览模式下，滚动鼠标滚轮可**以鼠标当前位置为中心**进行丝滑放大/缩小。
- **拖拽移动**：按住鼠标左键可自由拖动当前正在预览的图片。

### ⌨️ 快捷键操作
在图片全屏预览界面下，支持以下快捷键：

| 快捷键 | 功能 |
| :--- | :--- |
| `Esc` | 退出并关闭预览 |
| `+` / `=` | 放大 |
| `-` / `_` | 缩小 |
| `0` | 重置缩放比例和位置 |
| `F` | 自动适配当前窗口大小 (Fit) |
| `R` | 向右旋转 90 度 |

## 🛠️ 技术栈与架构

- **Manifest V3**：遵循最新一代的安全与性能 Chrome 扩展规范。
- **Vanilla JavaScript**：无任何第三方依赖（如 React/Vue），纯原生 JS 编写，插件体积做到极致轻量。
- **Shadow DOM**：确保预览器的 UI 结构代码（如 `contentScript.js` 中的结构）与宿主页面的 DOM 环境和样式完全隔离，避免任何冲突。

## 📝 目录结构

```text
├── background.js       # Service Worker 脚本，用于处理跨域、右键菜单等后台逻辑
├── contentScript.js    # 核心注入脚本，负责生成预览 UI 及处理所有的交互事件逻辑（缩放、拖拽、旋转等）
├── manifest.json       # Chrome 插件核心配置与清单文件
└── README.md           # 项目说明文档（本文档）
```
